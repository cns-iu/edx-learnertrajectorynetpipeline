edX Learner and Course Analytics and Visualization Documentation

March 21, 2019

This is an R Markdown documentation for the visualizations produced using data processing and analysis results of the edX Learner and Course Analytics portion of the pipeling. Each figure is a statistical graph that created using R Studio; numerous packages from Tidyverse, including plyr, stringr, ggplot2; the colorspace package was used to generate color palettes for the visualizations; RCurl for importing data from github.

The scripts use sample data generated by the scripts found at the edX Learner and Course Analytics and Visualization Pipeline GitHub repository. These data sets are provided in the respository’s Data directory. The sample data sets are briefly described below. A full description of each data set and field list are available for download at LINK.

Data A was created by the script edX-1-courseStructureMeta.R. The data is a CSV of the the course structure used to represent the content of the MITxPro course, Architecture of Complex Systems, Fall 2016.

Data E

Data E was generated by combining three outputs of script edX-6-moduleUseAnalysis.R. Script edX-6-moduleUseAnalysis.R calculates statistics for modules use of certified and non-certified studnets in MITxPro course, Architecture of Complex Systems, Fall 2016.

Data E combines the unq_stu, events, and totalTime data from module use analysis applied to three different list of students: 1. all students (data was taken from Data C), 2. students who are certificate, 3. students who were not certified. The first set of data was taken from Data C. The remaining two sets of data in Data E were generated from lists of student identifiers based on from an analysis of the cert_status field in Data D.
